# Code of Conduct
## Our Goal
We want this project to be welcoming, creative, and collaborative for anyone.

## Acknowledgement
By participating in this project you agree to follow this Code of Conduct.

## Be Respectful
- Treat others in the community with respect and kindness.
- Assume good intent. Feedback can be hard to hear sometimes. Try and assume the best.
- Disagree with ideas, not with people.

## Be Construcive
- Give feedback that is clear and specific.
- Explain the why behind feedback.
- Avoid personal attacks.

## Respect Creative Work / AI Policy
- Do not submit plagiarized or copyrighted work.
- AI generated text is not allowed to write the actual chapters.
- AI used to help generate documentation or summaries is acceptable.

## Keep the Space PG-13
The following space does not allow:

- Harassment or discrimination
- Threats, doxxing, or encouraging harm
- Sexual content
- Spam, trolling, or bad-faith disruptions

## Scope
This Code of Conduct applies to:

- Issues, pull requests, and discussions
- Commit messages and reviews
- Any official community spaces including the Discord and Reddit

## Reporting
If something happens that violates this Code of Conduct:

- Open an issue titled Conduct Report (if you are okay reporting publicly) **or**
- Contact a maintainer privately (recommended for sensitive situations)

Include:
- What happened
- Where it happened
- When it happened
- Any relevant screenshots or links to context

## Enforcement
Maintainers may take any action needed to protect the community, including but not limited to:
- Issuing warnings or bans
- Requesting changes or apologies
- Removing comments or content
- Closing issues or pull requests

## How We Handle New Situtations
We can't predict every scenario. Maintainers will interpret the Code of Conduct in good faith, prioritize community safety, and update this document as needed.
